# 🚌 BluVoy Travels

BluVoy Travels is a responsive travel agency website developed using **HTML, CSS, and JavaScript**. The website provides information about travel packages, destinations, gallery, contact details, and includes an interactive booking form and chatbot.

---

## 🌐 Features

- 🏠 Responsive Home Page
- 👨‍💼 About Us Page
- 🏖 Tour Packages
- 🖼 Travel Gallery
- 📞 Contact Page
- 📝 Registration Popup
- 💬 Simple Travel Chatbot
- ⭐ Customer Reviews
- 📧 Newsletter Section
- 📱 Mobile Responsive Design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## 🚀 How to Run

1. Download or Clone the repository.

```
git clone https://github.com/your-username/BluVoy-Travels.git
```

2. Open the project folder.

3. Double-click **index.html** or open it using **Live Server** in Visual Studio Code.
